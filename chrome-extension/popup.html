Installed

Open the Origami AI app, start a screen recording, then switch to the browser tab you want to capture.

This extension will stream browser-tab cursor, click, keypress, and scroll telemetry back to the app so it can generate zoom, pan, and follow effects from real in-page activity.

Browser pages only. Native apps and the OS still require visual capture without DOM-level telemetry.

chrome-extension/ is the unpacked folder to load in Chrome.
